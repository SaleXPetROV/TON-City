# TON-City Бизнес-Модель V2.0

## Обзор экономической системы

TON-City использует трёхуровневую экономическую систему с 21 бизнесом, где каждый бизнес имеет 10 уровней прокачки. Бизнесы взаимосвязаны через цепочки производства и потребления ресурсов.

### Налоговые ставки
| Tier | Налог | Описание |
|------|-------|----------|
| Tier 1 | 15% | Базовое производство ресурсов |
| Tier 2 | 23% | Переработка и производство |
| Tier 3 | 30% | Финансы и инфраструктура |

**Дополнительные налоги:**
- Оборотный налог: 2% с каждой транзакции на рынке
- Налог покровителю: 1%
- Комиссия мгновенного вывода: 1%

## Формулы расчёта

### Рост производства по уровням
```
Производство[Уровень N] = Производство[Уровень 1] × 1.5^(N-1)
```

### Рост потребления по уровням
```
Потребление[Уровень N] = Потребление[Уровень 1] × 1.45^(N-1)
```

### Стоимость апгрейда
```
Стоимость[Уровень N] = Базовая_стоимость × 1.6^(N-1)
```

### Влияние прочности на доход
```
Реальный_доход = Базовый_доход × (Прочность% / 100)
```

### Чистая прибыль
```
Чистая_прибыль = Доход - Расходы_на_ресурсы - Налог(Tier%) - Оборотный_налог(2%)
```

## Стратегии развития

### Для продвинутых (200+ TON)
1. Инвестируйте в **Gram Bank** или **Validator**
2. Создайте полную цепочку производства

4. Прокачивайте Tier 3 бизнесы до 7-10 уровня

## Риски и защита

### Полный склад
- Бизнес останавливается если склад заполнен на 100%
- Решение: Продавайте ресурсы на рынке или расширяйте склад

### 0% прочности
- Бизнес полностью останавливается
- Решение: Используйте ремкомплекты или вызывайте ремонт

### Нехватка ресурсов
- Бизнес работает вхолостую если нет входящих ресурсов
- Решение: Покупайте ресурсы на рынке или стройте бизнесы-поставщики

### Ночной распад (00:00 МСК)
- 10% ресурсов на складах теряется ежедневно в полночь
- Решение: Продавайте излишки перед полуночью
